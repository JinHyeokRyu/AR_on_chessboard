# AR_on_chessboard
원하는 그림이나 이미지를 chessboard 위에 띄워보세요! <br>
마치 3차원 현실 세계에 있는 것만 같은 느낌을 느끼실 수 있을겁니다!<br>

## Camera Pose Estimation

카메라 영상의 chessboard를 통해 카메라의 각도, 위치를 계산합니다.<br>
이 덕분에 여러분의 object를 3차원적으로 보여줄 수 있습니다!

## AR Object Visualization

여러분들이 chessboard 위에 띄우고자 하는 object의 정보(좌표값)를 가지고, 3차원에서 만나 볼 시간입니다!
